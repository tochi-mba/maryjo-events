# ADR-002: One-Page Section-First Information Architecture

## Status
Accepted

## Decision
Launch as a one-page site with anchored sections: Hero, Services, Approach, About, Event Types, Process Proof, and Contact.

## Rationale
Mary-Jo is an emerging freelance planner. A focused one-page site gives buyers a clear path from fit to inquiry without asking them to browse thin pages.

## Consequences
- `/` is the main page.
- `/thank-you` and `/privacy` are static supporting routes.
- Navigation uses real anchor links for crawlability and accessibility.
