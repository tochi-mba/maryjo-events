# ADR-004: Custom Components, Not a Heavy UI Framework

## Status
Accepted

## Decision
Use small custom Vue components instead of a broad UI framework.

## Rationale
The brand needs warm, personal, specific details. A generic UI kit would add weight and make the site feel less like Mary-Jo.

## Consequences
Core components are `Wordmark`, `SectionHeader`, `ServiceCard`, `ProcessStep`, `ContactForm`, and `GeneratedImage`.
