# ADR-003: Brand System as Design Tokens

## Status
Accepted

## Decision
Encode Mary-Jo's brand palette and typography as CSS variables and TypeScript constants.

## Tokens
- Berry: `#3d1a2e`
- Fuchsia: `#c24d7e`
- Blush: `#e8829c`
- Petal: `#f2dde8`
- Gold: `#f7c344`
- Display/headings/quotes/wordmark: Georgia
- Body, labels, UI: Arial

## Consequences
- Avoid cold grey and corporate navy tones.
- Use Gold only for small moments of emphasis.
- Do not use more than three brand colours in a single visual block.
