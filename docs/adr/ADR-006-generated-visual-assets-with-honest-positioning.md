# ADR-006: Generated Visual Assets with Honest Positioning

## Status
Accepted

## Decision
Use generated, brand-matched website imagery until real event photography is available.

## Rationale
The site needs visual richness without pretending Mary-Jo has a mature case-study archive.

## Consequences
- Do not use fake testimonials, fake client logos, or fabricated case studies.
- Visuals should support planning care, event detail, and warm corporate polish.
